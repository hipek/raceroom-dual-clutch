# RaceRoom Dual Clutch Setup Guide

This repository provides a guide to set up dual-clutch functionality in RaceRoom Racing Experience using a normal clutch pedal and a button on the steering wheel.

## Problem

Many racing wheels don't have extra analog paddles. This solution allows you to use a regular clutch pedal combined with a button on your wheel to achieve dual-clutch launching.

## Required Software

### vJoy (v2.2.2.0)

Virtual joystick driver for Windows.

- Download: https://github.com/BrunnerInnovation/vJoy

### Joystick Gremlin (v14.1)

A tool for configuring and managing joystick devices.

- Download: https://whitemagic.github.io/JoystickGremlin/download.html
- GitHub: https://github.com/WhiteMagic/JoystickGremlin
